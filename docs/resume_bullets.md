# 简历与面试表达

## 简历版本一：稳健版

```text
A 股多因子选股策略研究与回测系统 | Python, Pandas, NumPy, Matplotlib
- 构建基于价值、质量、动量、低波动和流动性的 A 股多因子选股框架，覆盖因子计算、预处理、有效性检验、组合构建和回测评估完整流程。
- 对因子进行缺失值填充、MAD 去极值、Z-score 标准化及行业/市值中性化处理，使用 IC、Rank IC、ICIR、分层回测和因子相关性分析评估因子有效性。
- 采用等权合成与滚动 Rank IC 加权两种多因子合成方法，每月调仓选取得分最高的股票构建等权组合，并引入交易成本和换手率分析。
- 输出年化收益、夏普比率、最大回撤、信息比率、超额收益、市场环境分阶段表现和行业/风格暴露分析，形成可复现策略报告。
```

## 简历版本二：结果导向版

在你跑出真实数据结果后，可以替换下面的占位符：

```text
- 在 2018-2024 年沪深 300 股票池回测中，滚动 Rank IC 加权组合取得 X% 年化收益、Y 夏普比率、Z% 最大回撤，相对基准实现 X% 年化超额收益。
```

不要在没有真实结果时写收益数字。面试官很容易追问数据源、回测区间、交易成本和是否存在未来函数。

## 面试讲解框架

1. 我先构建了五大类因子，并将所有因子统一为「数值越大越好」。
2. 每个调仓日做截面预处理，包括缺失值填充、MAD 去极值、Z-score 和行业/市值中性化。
3. 我不直接看最终收益，而是先用 IC、Rank IC 和分层回测验证因子有效性。
4. 多因子合成方面，我对比了等权方法和滚动 Rank IC 加权方法，后者只使用历史 IC，避免未来函数。
5. 组合构建使用月度调仓、Top N、等权和行业中性选股，并加入交易成本。
6. 结果分析包括收益风险指标、最大回撤、月度收益、行业暴露、风格暴露、市场环境分阶段表现和参数敏感性。
