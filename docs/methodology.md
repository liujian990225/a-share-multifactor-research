# 方法论说明

## 1. 研究目标

本项目构建一个 A 股多因子选股策略研究框架。策略在每个月调仓日计算截面因子，对股票进行综合打分，并选择得分最高的股票构建组合。评价重点包括因子有效性、组合收益风险特征、超额收益稳定性、市场环境适应性和组合暴露。

## 2. 股票池与过滤

推荐股票池：沪深 300 或中证 500 成分股。

基础过滤：

- 剔除 ST 股票；
- 剔除上市不足 180 天股票；
- 剔除停牌或不可交易股票；
- 剔除过去 20 日平均成交额过低的股票；
- 剔除关键字段缺失过多的样本。

## 3. 因子体系

### 价值因子

- `value_ep = 1 / PE_TTM`
- `value_bp = 1 / PB`
- `value_sp = 1 / PS_TTM`
- `value_dividend = dividend_yield`

### 质量因子

- `quality_roe`
- `quality_roa`
- `quality_gross_margin`
- `quality_net_margin`
- `quality_low_debt = -debt_to_asset`

### 动量因子

- `momentum_20d`
- `momentum_60d`
- `momentum_120d`

### 低波动因子

- `low_vol_20d = -vol_20d`
- `low_vol_60d = -vol_60d`
- `low_mdd_60d = -max_drawdown_60d`

### 流动性因子

- `liquidity_log_amount = log(1 + amount_20d)`
- `liquidity_low_amihud = -amihud_20d`

所有因子都统一为「数值越大越好」。

## 4. 因子预处理

每个调仓日单独进行截面处理：

1. 缺失值处理：优先使用行业中位数填充，再使用全市场中位数填充。
2. 极值处理：使用 MAD 去极值。
3. 标准化：使用 Z-score 标准化。
4. 中性化：对行业哑变量和市值因子做横截面回归，取残差作为中性化后的因子。
5. 再标准化：中性化后重新 Z-score。

## 5. 因子检验

- IC：因子值与下一期收益的 Pearson 相关系数。
- Rank IC：因子排名与下一期收益排名的 Spearman 相关系数。
- ICIR：IC 均值 / IC 标准差。
- IC 胜率：IC 大于 0 的月份占比。
- 分层回测：按因子分位数分组，观察高分组是否稳定跑赢低分组。
- 因子相关性：检查因子冗余和多重共线性。

## 6. 多因子合成

### 等权合成

先对同一类别内的因子取平均，再按类别权重合成：

```text
score_equal = 0.25 * value
            + 0.25 * quality
            + 0.20 * momentum
            + 0.15 * low_volatility
            + 0.15 * liquidity
```

### 滚动 Rank IC 加权

对每个因子使用过去 12 个月 Rank IC 均值作为动态权重。若历史样本不足，则退化为等权。权重只使用当前调仓日前已能观察到的 IC，避免未来函数。

## 7. 组合构建

- 调仓频率：月度；
- 执行价格：默认下一交易日收盘价近似；
- 持仓数量：Top 30 或配置中的 Top N；
- 权重方式：等权；
- 行业约束：可按行业比例进行 Top N 选股，降低行业押注；
- 交易成本：默认单边 10 bps。

## 8. 进阶分析

- 市场环境分阶段：根据基准指数过去 120 日涨跌幅和均线位置划分牛市、熊市、震荡市。
- 行业暴露：统计组合在各行业上的权重分布。
- 风格暴露：统计组合在市值、估值、质量、动量、波动率上的平均暴露。
- 参数敏感性：比较不同持仓数量和交易成本下的绩效变化。
