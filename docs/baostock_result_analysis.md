\# BaoStock 真实数据回测结果分析



\## 1. 项目运行概况



本项目已经完成基于 BaoStock 免费数据源的真实 A 股数据回测流程。当前运行结果说明，项目的数据下载、因子构造、因子预处理、IC 检验、组合构建、机器学习 Alpha、因子择时、市场状态分析和参数敏感性分析等模块均已成功跑通。



当前结果更适合作为“真实数据流程验证版”和“GitHub 项目展示版”，暂不应直接视为可实盘交易策略结论。



\## 2. 当前实验设置



当前回测设置大致如下：



| 项目 | 设置 |

|---|---|

| 数据源 | BaoStock |

| 市场 | A 股 |

| 回测区间 | 2021-01-01 至 2024-12-31 |

| 调仓频率 | 月度调仓 |

| 股票池规模 | 约 50 只股票 |

| 组合持仓 | Top 20 |

| 交易成本 | 单边 10 bps |

| 策略类型 | Long-only 多头选股 |

| 机器学习模型 | Random Forest / Walk-forward |

| 输出报告 | reports/baostock\_run/ |



当前 BaoStock 版本主要使用行情、估值和量价数据构造因子，包括价值因子、动量因子、低波动因子、流动性因子和市值相关特征。由于 BaoStock 当前版本没有完整接入高质量财务指标，因此质量因子模块在代码层面保留，但在当前实验中没有充分发挥作用。



\## 3. 策略表现概览



当前报告中对比了四类策略：



| 策略 | 含义 |

|---|---|

| score\_equal | 等权多因子合成 |

| score\_ic\_weighted | 滚动 IC 加权多因子合成 |

| score\_ml\_alpha | 机器学习 Alpha 预测 |

| score\_ic\_forecast | 因子 IC 预测择时 |



当前结果显示，`score\_ic\_forecast` 表现最好，其次是 `score\_ic\_weighted` 和 `score\_ml\_alpha`，而简单等权多因子策略表现相对较弱。



主要结论是：



1\. 动态因子权重优于固定等权合成；

2\. 滚动 IC 加权能够改善收益和回撤；

3\. 机器学习 Alpha 有一定提升，但在当前小样本下不宜过度解读；

4\. 因子 IC 预测择时策略在当前样本中表现最好，说明因子有效性具有明显时变特征。



\## 4. 因子有效性分析



\### 4.1 低波动因子表现最稳定



在当前 BaoStock 样本中，低波动因子表现最稳定。`low\_vol\_20d` 和 `low\_vol\_60d` 的 Rank IC 均值为正，且胜率较高。



这说明在当前样本期内，市场更偏好防御型、波动较低的股票。低波动因子不仅提升了策略收益，也对回撤控制有一定帮助。



可以解释为：



> 在 2021-2024 年的 A 股样本中，市场整体波动较大，低波动股票表现出更强的防御属性，因此低波动因子具有较好的预测能力。



\### 4.2 价值因子有一定效果



价值因子中，`value\_bp`、`value\_ep` 和 `value\_sp` 整体表现为正向，说明低 PB、低 PE、低 PS 的股票在当前样本中具有一定超额收益特征。



但价值因子的稳定性弱于低波动因子。后续如果接入更完整的财务数据，可以进一步加入盈利质量、现金流和成长性指标，避免单纯低估值带来的价值陷阱问题。



\### 4.3 动量因子表现为负



当前样本中，`momentum\_20d`、`momentum\_60d` 和 `momentum\_120d` 的 Rank IC 均值为负，说明简单价格动量因子在该样本中没有表现出趋势延续特征，反而更接近短中期反转。



这可能与以下因素有关：



1\. A 股市场短期反转较明显；

2\. 样本区间包含较多震荡和下跌行情；

3\. 热门股票在阶段性上涨后容易回落；

4\. 当前股票池较小，个股影响较大。



因此，后续可以考虑把动量因子改造为反转因子，或者根据市场状态动态决定动量因子的方向。



\### 4.4 流动性因子方向需要谨慎解释



当前样本中，流动性相关因子表现为负。这说明成交额较高、流动性较好的股票在当前小样本中未来收益反而偏弱。



但这并不意味着流动性越差越好。更合理的解释是：



1\. 当前股票池较小，结果容易受个股影响；

2\. 热门高成交股票可能存在过热回落；

3\. 样本可能偏向某些风格或行业；

4\. 流动性因子更适合作为风险过滤条件，而不一定直接作为收益因子。



后续建议将流动性因子主要用于剔除低成交额股票，而不是简单地作为收益预测因子。



\## 5. 机器学习 Alpha 分析



当前 Level 3 项目中引入了机器学习 Alpha 模型，使用 walk-forward 框架进行训练和预测。



其核心流程是：



1\. 每个调仓日只使用历史数据训练模型；

2\. 使用历史因子值预测未来收益；

3\. 在当前调仓日生成股票 Alpha 得分；

4\. 根据预测得分构建组合；

5\. 避免使用未来信息，降低未来函数风险。



当前机器学习 Alpha 策略表现优于等权多因子，但没有超过 IC 预测择时策略。



这说明：



1\. 机器学习模型可能捕捉到了一些非线性关系；

2\. 动量、市值、低波动和流动性特征对模型影响较大；

3\. 但当前样本股票数量较少，机器学习结果需要谨慎解释；

4\. 机器学习并不一定优于传统因子方法，尤其在样本较小、特征有限时。



后续如果扩大股票池至 300 只以上，并延长回测区间，机器学习 Alpha 的结果会更有参考价值。



\## 6. 市场状态分析



\### 6.1 熊市阶段



在熊市阶段，策略整体表现优于基准，尤其是 IC 预测择时策略表现较好。虽然策略可能仍有亏损，但相比基准跌幅更小，体现出一定防御属性。



这与低波动因子表现较强的结论一致。



\### 6.2 牛市阶段



在牛市阶段，策略明显跑输基准。



这说明当前策略偏防御，在强趋势上涨市场中进攻性不足。低波动、价值和防御属性较强的因子在牛市中往往不如高弹性成长股和趋势股。



后续可以考虑加入成长因子、盈利修复因子、景气度因子或趋势增强因子，以改善牛市阶段的表现。



\### 6.3 震荡市阶段



在震荡市阶段，滚动 IC 加权和 IC 预测择时策略表现较好。



这说明动态因子权重在风格轮动较快的市场中更有优势，能够根据近期因子表现调整策略暴露。



\## 7. 参数敏感性分析



参数敏感性分析显示，集中持仓可能提升收益，但也会提高回撤和样本依赖。



例如，当 Top N 较小时，机器学习 Alpha 策略收益可能更高，但组合集中度也更高，风险更大。



因此，不能简单认为 Top 10 或 Top 20 一定最优，需要结合：



1\. 股票池规模；

2\. 最大回撤；

3\. 换手率；

4\. 交易成本；

5\. 样本外稳定性；

6\. 策略容量。



当前股票池较小，因此 Top N 的选择对结果影响较大。后续扩大股票池后，需要重新进行参数敏感性测试。



\## 8. 当前结果的主要局限



\### 8.1 股票池较小



当前股票池约 50 只，实际有效股票数量可能更少，而组合 Top N 为 20。



这意味着每个月组合几乎买入了大部分可选股票，选股区分度不足。



更合理的设置是：



\- 如果股票池为 50 只，Top N 可设为 5 或 10；

\- 如果 Top N 为 20，股票池建议至少 100 到 200 只；

\- 如果做正式量化研究，股票池建议扩展到 300 只以上。



\### 8.2 行业数据缺失



当前 BaoStock 数据中行业字段基本为 Unknown，因此行业中性化和行业暴露分析没有真正发挥作用。



后续可以使用 AKShare、申万行业分类、东方财富行业分类等数据补充行业标签。



\### 8.3 质量因子缺少真实财务数据



当前 BaoStock 版本没有完整接入 ROE、ROA、毛利率、净利率、资产负债率等高质量财务字段。



因此，本次真实数据实验主要验证的是行情、估值和量价因子框架，而不是完整的价值、质量、动量、低波动、流动性五因子体系。



\### 8.4 机器学习样本不足



当前每个月有效股票数量较少，对机器学习模型来说样本偏小。



虽然机器学习 Alpha 的结果较好，但需要在更大股票池、更长时间区间上验证，避免过拟合或样本偶然性。



\### 8.5 免费数据源限制



BaoStock 适合免费跑通真实数据流程，但在行业分类、完整财务指标、指数历史成分股等方面不如专业数据源完整。



后续如果接入 Tushare 高权限、聚宽、Wind 或 RiceQuant，研究结果会更严谨。
